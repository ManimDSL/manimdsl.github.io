Introduction
-----------------

Welcome to Manim DSL's documentation!

Manim DSL is a domain specific language that helps educators create animations for teaching students about data structures
and algorithms.

Here is an example:

.. code:: javascript

    let stack: Stack = new Stack;
    stack.push(5);
    stack.pop();

This would produce the video:

.. raw:: html

    <div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden; max-width: 100%; height: auto;">
        <iframe src="../media/intro.mp4" frameborder="0" allowfullscreen style="position: absolute; top: 0; left: 0; width: 80%; height: 80%;"></iframe>
    </div>
